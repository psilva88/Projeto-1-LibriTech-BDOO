# Projeto-1-LibriTech-BDOO
O grupo foi contratado para desenvolver o LibriTech (vocês podem substituir o nome por outro), um sistema robusto para o gerenciamento de uma biblioteca universitária. Neste projeto, o MySQL não será apenas um depósito de dados; ele atuará como o "cérebro" que protege as regras de negócio e a segurança.
